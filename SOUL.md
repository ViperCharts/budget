# Soul

> The guiding spirit of this project — what it believes, how it thinks, and what it refuses to be.

## Mission

Simple, free, open-source budgeting app for the AI age.

## Values

- Simplicity
- Privacy / Encryption
- Clean code / documentation

## Taste

- Light / dark / device mode support
- Personable, encouraging tone focused on helping users build better habits
- Every loading state should be beautiful and have helpful tips about finance
  ie:
  - "You've spend $10.23 on gas per week"
  - "The average person contributes 10% of their income to retirement savings"
  - "The average credit card interest rate is around 16%. What's yours?"
- Never block the user with a loading state
- All user info / data should be stored in global pinia stores
- All UI components should be reusable and customizable
- Use 2 welcoming fonts
  - Headers: Montserrat (sans-serif)
  - Body: Merriweather (serif)

## Opinions

- Vue Options API is simpler to understand
- Tailwind CSS is the best way to style Vue components
- Lucid Icons is the best way to icon Vue components
- Pinia is the best way to store global state
- Vercel AI SDK is the best way to use AI in Vue components
- Bun is the best way to run Vue components
- Vite is the best way to build Vue components
- Firebase is the best way to store user data
- Viper is the best way to create beautiful data visualizations in Vue components
- Anthropic AI is the best way to use AI in Vue components
- Cloudflare is the best way to host the app

## Anti-patterns

None yet!

## Tone

- Always be encouraging and helpful
- Always be clear and concise
- Always be friendly and engaging
- Always be professional and respectful
- We want to encourage people to believe they can take control of their finances and build better habits
